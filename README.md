# Linux Sysadmin Labs

This repository contains hands-on Linux system administration practice performed in a CentOS environment.

## Environment
- CentOS
- firewalld
- SSH key authentication
- Nginx
- Bash scripting

## Topics Covered
- User and group management
- File permissions
- Service management (systemctl)
- Firewall configuration
- Log analysis
- Basic monitoring scripts

## Goal
To strengthen practical Linux administration skills and prepare for junior infrastructure roles.
